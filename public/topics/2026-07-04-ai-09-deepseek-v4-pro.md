## 一行サマリー
DeepSeek V4 Pro は MIT ライセンス・100万トークンのコンテキストを備え、総合スコア 87 でオープンウェイトモデルの首位とされる（要確認）。GPT-5.5 級の性能を大幅な低コストで狙える点が注目。

## 🔰 初学者向け（何が・なぜ重要か）
これまで「最も賢いAI」は OpenAI や Anthropic のような非公開モデルが独占していました。使うにはその会社の API を借りるしかなく、中身は覗けません。一方 DeepSeek V4 Pro は「オープンウェイト」、つまりモデルの重み（頭脳そのもの）が公開されており、しかも MIT という非常に緩いライセンスで、商用にもほぼ自由に使えます。

さらに「100万トークンのコンテキスト」とは、本一冊分どころか数冊分をまとめて読み込ませても文脈を保てる、という桁違いの記憶量です。これがトップクラスの性能とセットで、しかも自前サーバーで安く動かせるとなれば、「高いクローズド API に頼らない」選択肢が現実味を帯びます。

## 🧠 専門的解説（技術詳細・設計・使いどころ）
### 技術的詳細（何が変わったか）
benchlm によれば DeepSeek V4 Pro は MIT ライセンス、1M トークンコンテキスト、総合スコア 87 でオープンウェイト首位。加えて llm-stats 系の比較では「DeepSeek V4 が GPT-5.5 級の性能を約34倍安く提供」とされる。これら数値は二次情報由来のため公式リリースでの**要確認**。1M コンテキストと最上位級スコアの両立が最大の売り。

### 仕組み / 背景（なぜこう設計されたか）
DeepSeek はコスト効率の高い学習・推論（MoE アーキテクチャや効率的な学習手法）で知られ、少ない計算資源で高性能を狙う設計思想を持つ。MIT ライセンスでのオープン公開は、エコシステム普及とデファクト化を狙う戦略。オープン対クローズドの性能差が縮小する象徴的存在で、コスト面ではクローズド勢を大きく下回る点がトレードオフの均衡を変えつつある。

### 実務での使いどころ・移行の勘所
超長文の一括処理（大規模コードベース、長い契約書群、書籍横断）や、データを外部 API に出したくない自前ホスト運用に最適。MIT ライセンスゆえ商用・改変・再配布のハードルが低い。一方、自前運用は GPU コスト・運用負荷・安全対策を自社で持つ必要があり、クローズド API のマネージド運用との総保有コスト比較が肝。軽量版の V4 Flash と用途で棲み分けるとよい。

## 📖 用語解説（このトピックとのつながり）
- **オープンウェイト**: モデルの重みが公開された形態。自前ホスト・改変が可能。本記事の核。
- **MIT ライセンス**: 極めて緩い許諾。商用・改変・再配布がほぼ自由。導入障壁を下げる。
- **1M コンテキスト**: 100万トークンを一度に扱える記憶量。長文一括処理の武器。
- **MoE（Mixture of Experts）**: 一部の専門家サブネットのみ活性化し効率化する構造。DeepSeek の低コストの背景（要確認）。

## 影響範囲 / 推奨アクション
コスト最適化やデータ主権を重視する企業・開発者に強く影響。クローズド API から自前ホストへの移行余地を、GPU コスト・運用負荷を含む総保有コストで評価すること。スコア・価格比は公式ソースで裏取りを。
