# ソニー Optical Disc Archive Software CVE-2026-50255 — インストーラの不適切な権限設定でSYSTEM昇格

> トピック: 脆弱性ニュース ｜ 出典: JVN（JVN#79926428）/ JPCERT/CC ｜ 種別: 不適切な既定の権限設定（CWE-276）

## 一行サマリー

ソニー製「Optical Disc Archive Software for Windows」のインストーラに、フォルダ/ファイルのアクセス権が緩く設定される欠陥（CVE-2026-50255）があり、一般ユーザーが細工によりSYSTEM権限で任意コードを実行できる。ローカル権限昇格型で、端末利用者による権限奪取に悪用されうる。

## 🔰 初学者向け（何が・なぜ問題か）

ソフトをインストールすると、プログラムのファイルがパソコンの中の決まった場所に置かれます。本来こうしたファイルは「管理者だけが書き換えられる」よう厳重に守るべきものです。ところが今回のインストーラは、置いた場所の鍵を緩く設定してしまい、**一般ユーザーでもそのファイルを差し替えられる**状態にしていました。

たとえるなら、会社の重要書類を「誰でも開けられる引き出し」に入れてしまったようなもの。後からその引き出しの書類を悪意ある社員がこっそり偽物にすり替えると、次にそれを“偉い人（SYSTEM＝Windowsで最も強い権限）”が開いて実行したとき、すり替えた偽の命令が最強権限で動いてしまいます。

結果として、普通の利用者権限しか持たない人が、パソコンを完全にコントロールできる管理者級の権限を手に入れてしまいます。これは「権限昇格（けんげんしょうかく）」と呼ばれ、外部から直接侵入する穴ではありませんが、すでに端末を使える人や、いったん入り込んだマルウェアが“格上げ”するのに使われます。

## 🧠 専門的解説（技術詳細・メカニズム・検知対応）

### 技術的詳細（種別・前提条件）

- **CVE/JVN**: CVE-2026-50255 / JVN#79926428（JVNDB-2026-000084、JPCERT/CC 調整）。
- **CVSS**: 6.7（Medium）。種別は Incorrect Default Permissions（CWE-276、不適切な既定の権限設定）。
- **前提条件**: 対象端末でのローカル操作（一般ユーザー権限）。リモート単独の侵入口ではない。
- **影響 / 対象**: Optical Disc Archive Software for Windows **5.5.3 以前**。悪用されると **SYSTEM権限で任意コード実行**。
- **修正**: 開発者（ソニー）提供の最新インストーラを使用すること。公開日 2026-06-16。
- **悪用状況**: 本稿時点で実環境悪用の報告は要確認。

### メカニズム（攻撃が成立する仕組み）

1. **緩い既定ACL**: インストーラが作成するディレクトリ/ファイルに、一般ユーザーでも書き込み可能なアクセス制御（ACL）が設定される。本来「管理者のみ書き込み可」であるべき領域が、信頼境界を越えて低権限ユーザーに開いている。
2. **書き換えプリミティブの獲得**: 低権限の攻撃者は、そのフォルダ内の実行ファイルやDLL、設定を**自分の用意したものに置き換える**ことができる。これが「特権領域への任意書き込み」というプリミティブ。
3. **高権限プロセスによる実行**: 当該ソフトのサービスやスケジュール処理、アップデータがSYSTEM等の高権限で起動する際、すり替えられた実行ファイル/DLLを読み込む（DLLハイジャックや実行ファイル置換の典型）。これにより攻撃者コードがSYSTEM権限で走る。
4. **被害への橋渡し**: SYSTEM奪取＝端末の完全掌握。永続化、セキュリティ製品の無効化、認証情報窃取、横展開の起点となる。

一般原理として、これは「インストール時のACL設定不備（least privilege違反）」が「特権実行パスへの介入」に橋渡しされるローカル権限昇格の典型。書き込み可能ディレクトリから高権限プロセスがバイナリをロードする構図（writable path ＋ privileged load）が鍵で、TOCTOUやDLL検索順序の問題と同系統の被害に至る。武器化手順やPoCはここでは扱わない（概念レベル）。

### 検知・暫定緩和・恒久対策

- **検知の着眼点**: 当該インストール先ディレクトリのACL（一般ユーザーの書き込み権限有無）、配下の実行ファイル/DLLの予期せぬ変更・タイムスタンプ更新、SYSTEM権限プロセスによる非標準パスからのモジュールロード。
- **暫定緩和**: インストールディレクトリのACLを管理者のみ書き込み可に是正、一般ユーザーの書き込み権限を剥奪、当該ソフトのサービス/アップデータの権限と起動経路の確認。
- **恒久対策**: 開発者提供の**最新インストーラで再インストール／更新**し、適切な権限で再配置。標準のProgram Files配下など保護された領域へのインストールを徹底。

## 📖 用語解説（このニュースとのつながり）

- **不適切な既定の権限設定（CWE-276）**: インストール時などに付くアクセス権が緩すぎる欠陥。本件の根本原因。
- **ACL（アクセス制御リスト）**: 「誰がそのファイルを読み書きできるか」の設定。本件はこれが緩く、一般ユーザーが書き換え可能だった。
- **SYSTEM権限**: Windowsで最も強い権限。本件の最終的な奪取対象。
- **権限昇格（ローカル）**: 端末内で低権限から高権限を奪う攻撃。本件はその典型。
- **DLLハイジャック／実行ファイル置換**: 高権限プロセスが読み込むモジュールを攻撃者が差し替える手口。本件の橋渡しに該当。

## 影響範囲 / 推奨対応

Optical Disc Archive Software for Windows 5.5.3 以前を導入した端末。映像・放送・アーカイブ用途で業務端末に入っていることがある。リモート侵入口ではないが、共有端末や侵入後の権限昇格に直結するため放置は禁物。**最新インストーラで更新**し、インストール先ディレクトリのACLを管理者のみ書き込み可に是正する。
